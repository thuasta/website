# 使用终端

终端（Terminals），也称为命令行（command lines）或控制台（consoles），允许我们在不使用图形用户界面的情况下完成和自动化计算机上的任务。

- [你必须知道的 50+ Linux 命令](https://www.digitalocean.com/community/tutorials/linux-commands)

:::
