# Python

Python 是一种广泛使用的编程语言，结合了强类型和动态类型的特点。作为解释型语言，Python 代码在编写后可以立即执行，无需提前编译。Python 的语法灵活，支持函数式（functional）、过程式（procedural）和面向对象（programmatic）等多种编程范式。
